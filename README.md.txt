# IoT Data Processing Pipeline

A Spring Boot application to register IoT devices, ingest sensor data, and analyze it in real time with WebSocket dashboards, alerts, and analytics.

## Features
- Register and manage IoT devices
- Real-time data ingestion and broadcasting
- Device simulator with scheduled data push
- WebSocket-based dashboard
- Alerts, anomalies, trends and stats

## How to Run
1. Clone the repo
2. Open in IntelliJ or VS Code
3. Run `mvn spring-boot:run` or `./mvnw spring-boot:run`

## Tech Stack
- Java + Spring Boot
- WebSocket
- PostgreSQL / H2
- Chart.js (frontend)
- Optional: Kafka, Docker
